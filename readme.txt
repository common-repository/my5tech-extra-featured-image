== Description ==
= Extra featured image =

You need a extra featured image for posts, pages and/or custom post types? Then this plugin is for you!

It enables extra featured image for all post types and comes with a shortcode [my5tech_extra_image] for displaying your extra image on posts, pages and/or custom post types.

= Features =

1. Add an extra featured image as you need
2. Add the featured image to any post type (post, page or even custom post types)

== Installation ==
1. Unzip and upload the `extra-featured-image` directory to the plugin directory ('/wp-content/plugins/')
2. Activate the plugin through the 'Plugins' menu in WordPress
3. And then edit any post/page or custom post type an another metabox appear just below the Featured Image Box on right side.
4. After uploading and published post use shortcode [my5tech_extra_image] to display extra image.
5. Feel free to ask if you have problems with this plugin. Feature requests are welcome too!
